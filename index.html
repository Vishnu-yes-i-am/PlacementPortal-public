PlacementPortal

Introduction

The College Placement Portal is a fully functional web application developed for our college to streamline the placement process for students and companies alike.

The landing page of the College Placement Portal, showcasing key features.

Features

  • Student registration and login
  • Company registration and login
  • Job posting by companies
  • Job search and application by students
  • Admin panel for managing users, companies, and job postings
  • Email notifications for important events (e.g., successful job application)
  • Inbuilt view/edit or download resume for students
  • Red Flag Management to ensure fairness in the college placement process

Installation

    
$ npm install

Technologies Used

  • Frontend: HTML, CSS, JavaScript, FontAwesome5
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Email: Nodemailer
  • UI Library: Bootstrap
  • Miscellaneous: Google Calendar API, Pdflatex
